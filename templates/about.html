ABOUT

이미지 변환 처리는 Ai 머신러닝 기술과 여러분의 이미지로 완성됩니다.

[ YouHi ]는 유명 미술관에서 만날 수 있는 명화 느낌을 바로 내 생활에 적용하는 sns 웹 서비스입니다.
모든 이미지를 명화로 만들고, 우리의 일상을 명화로 기록하고, 자유롭게 공유해 보세요.

[ YouHi ] 탄생, AI 웹개발 트랙 1기 이미지 변환 팀프로젝트.
곽주혁: 서비스 첫 인상을 좌우하는 회원가입, 로그인, 로그아웃, 모달로 상세 게시물 보기 구현.
김현두: 유저 모두를 이어주는 메인페이지, 마이페이지, 상단바, 프로필 변경, 좋아요 기능 구현.
신동석: 이미지 유화 변환 처리를 실시간으로 미리 보여주는 게시물 업로드 기능 구현.
구현정: TensorFlow Hub 모델로 이미지 변환 작업 진행하여 OPEN API 구축.
이제 당신의 유화로 일상을 공유하세요!
